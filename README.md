## Overview
The **Revenue Insights Dashboard** is a data analytics project that analyzes and visualizes revenue data to uncover trends, key metrics, and future revenue patterns. It provides interactive insights to support data-driven business decisions.

---

## Key Features
- Revenue trend analysis  
- KPI metrics (Total Revenue, Average Revenue, Growth)  
- Interactive filters and visualizations  
- Revenue forecasting using rolling averages  
- Clean, business-focused dashboard  

---

## Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib
- Streamlit  
- CSV Data
